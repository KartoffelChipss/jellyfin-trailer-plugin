Local Trailers

Downloads YouTube trailers to disk as local -trailer.mp4 files in a trailers/ folder next to each media item, so every Jellyfin client can play them natively.

yt-dlp

Download and maintain the yt-dlp binary automatically. Detected version: …
Leave blank to use the managed binary.
Leave blank to use the Jellyfin-bundled ffmpeg (used by yt-dlp for merging formats).

Download settings

yt-dlp format selector. Default picks up to 1080p H.264+AAC.
HTTP/SOCKS5 proxy for yt-dlp (e.g. socks5://127.0.0.1:1080).
Space-separated extra arguments passed to yt-dlp (e.g. --cookies /path/to/cookies.txt).
How many trailers to download in parallel during the scheduled task.
When enabled, trailers that have already been downloaded will be re-downloaded.
